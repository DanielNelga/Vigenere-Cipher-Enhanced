Veginere Cipher

The Vigenère cipher is a polyalphabetic substitution cipher that encrypts text by using a keyword to apply a different shift to each letter of the message. The keyword is repeated across the length of the message so that every letter has a corresponding key character, and that key character determines how far the letter is shifted along the alphabet. Unlike a Caesar cipher which uses the same shift for every letter, the varying shifts of the Vigenère cipher mean that the same letter can encrypt to different characters depending on its position, making it significantly harder to crack through frequency analysis. Decryption works by simply reversing the process, shifting each letter backward by the same key values. The main weakness of the cipher is that if the key is short or predictable, an attacker who determines the key length can break the message down into groups and crack each one independently, which is why key length and complexity have a direct impact on how secure the encryption actually is.
